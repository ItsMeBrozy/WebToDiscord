# Discord Web Bridge 🚀

A real-time bridge that connects a Discord channel to a modern web dashboard.

## Setup for Hugging Face
1. Ensure the **SDK** is set to **Docker** in your Space settings.
2. Add your `DISCORD_TOKEN` and `CHANNEL_ID` to the **Secrets** section in the Space Settings.
3. The Space will automatically build using the included `Dockerfile`.

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
